# 🤖 InsureYouAI

Sigorta dünyasını modern yapay zekâ teknolojileriyle buluşturan,  
çoklu model orkestrasyonu üzerine kurulu deneysel bir platform.

Geliştirme: **.NET 10**

![AI](https://img.shields.io/badge/AI-Multi--Model-blue)
![.NET](https://img.shields.io/badge/.NET-Core-purple)
![n8n](https://img.shields.io/badge/Automation-n8n-orange)

---

## 🌠 Proje Vizyonu

Tek bir modele bağımlı kalmadan, farklı AI servislerinin güçlü yönlerini birleştirerek  
kolektif çalışan bir yapay zekâ ekosistemi oluşturmak.

- Daha doğru analiz  
- Daha güçlü karar mekanizması  
- Daha güncel veri  

---

## ⚡ Özellikler

### 🔀 Multi-AI Orchestration
Farklı LLM çıktıları tek akışta birleşir.

### 🧠 Context Analysis
Karmaşık sigorta poliçeleri derin analiz edilir.

### 🌐 Real-Time Data
Güncel veri entegrasyonu sağlanır.

### 🔊 Voice Output
Metin → doğal ses dönüşümü (TTS)

### 🔄 n8n Automation
Workflow orchestration ve API yönetimi

---

## 🧙‍♀️ AI Ekosistemi

| Servis | Görev |
|------|------|
| OpenAI | Ana karar motoru |

| Claude | Detaylı analiz |
